Toolbar : baris yang berisi kumpulan gambar/simbol yang mewakili suatu perintah dalam menu

Titlebar : berisikan nama program aplikasi dan nama file yang sedang aktif. Baris judul ini digunakan untuk memindahkan jendela ke posisi lain yang dinginkan(digeser dengan mouse)

Workspace area : daerah yang digunakan untuk melakukan pengetikan dokumen.

32. Mengapa disebut ikon pendukung di Microsoft Word?​


Jawaban:

MENGENAL IKON PENDUKUNG SOFTWARE PENGOLAH KATA

Simbol perintah atau ikon tidak hanya berbentuk tombol (button) pada jendela Word. Kombinasi tombol pada keyboard bisa diangap sebagai ikon karena fungsinya sama.

A.  MENAMPILKAN IKON PENDUKUNG

CARA 1  :

·         Buka Microsoft Word pada komputer kalian.

·         Klik menu Customize Quick Access Toolbar, maka akan muncul pilihan toolbar seperti gambar di samping.

·         Klik pilihan toolbar yang ingin kamu tampilkan.

CARA 2 :

·         Klik tombol Office Button.

·         Pilih dan klik perintah Word Options sehingga muncul kotak dialog Word Options.

·         Pilih dan klik perintah Customize sehingga muncul kotak dialog.

·         Pilih toolbar yang ingin ditampilkan. Klik Add, klik Ok

B.  IKON PENDUKUNG SOFTWARE PENGOLAH KATA

1.     IKON NUMBERING

·         Letakkan kursor di tempat yang akan diberi nomor.

·         Klik ikon Numbering  pada ribbon Home dalam group Paragraph.

Nomor akan tertulis secara otomatis pada lembar kerja.

Perintah penomoran juga bisa kamu tampilkan dengan langkah – langkah berikut :

·         Seleksi teks yang ingin diberi Numbering.

·         Klik ribbon Home. Pada group Paragraph pilih Numbering.

·         Jika kamu hendak memilih beberapa tipe nomor lebih lanjut, klik Define New number Format

·         Pada kotak dialog Define New number Format, ada beberapa pilihan, diantaranya :

o   Number Style, memilih model penomoran.

o   Font, memilih jenis font dan mengatur beberapa efek tertentu.

o   Number format, mengatur format nomor.

o   Alignment, mengatur perataan nomor.

o   Preview, melihat hasil

·         Klik Ok

Nomor akan tertulis secara otomatis.

2.    MENGOLAH GAMBAR

Program pengolah kata MS Word 2007 tidak hanya digunakan untuk membuat teks. Kamu bisa menggunakannya untuk mengolah gambar karena dilengkapi dengan fasilitas Drawing ( Menggambar ). Ikon apa saja yang ada di dalamnya? Penasaran, bukan? Sekarang, simaklah penjelasan berikut dengan cermat!

·         Klik ribbon Insert, pada group Illustration terdapat berbagai pilihan untuk membuat / menyisipkan gambar.

3.    INSERT CHART ( GRAFIK )

Perintah untuk membuat grafik dapat kamu temukan pada ribbon Insert.

Caranya :

·         Klik ribbon Insert

·         Pada group Illustrations, klik ikon Chart, jika kamu benar maka akan muncul tampilan seperti berikut :

4.    INSERT TABLE

Pada pelajaran kali ini, kita kembali mempelajari ikon dan beberapa fasilitas untuk membuat table. Ikutilah penjelasan berikut dengan baik :

Cara membuat table dengan ikon Draw Table

Kamu bisa membuat tabel pada komputer seperti menggambar dengan pensil. Caranya :

·         Aktifkan ribbon Insert pada group Tables pilih perintah untuk membuat tabel dengan meng-klik ikon Table.

·         Klik Ikon Draw, kursor akan berbentuk seperti pensil dan siap digunakan untuk menggambar tabel.

·         Click and drag mouse ke kanan

·         Lepaskan tombol kiri mouse, Klik!

Selanjutnya, bagilah kotak menjadi beberapa kolom. Caranya sama seperti diatas, tetapi dengan gerakan dari atas ke bawah. Jika kamu menggerakkannya dari kiri ke kanan, maka akan muncul bari baru.
