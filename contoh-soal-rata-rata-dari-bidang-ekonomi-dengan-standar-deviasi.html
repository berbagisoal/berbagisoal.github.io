Kode program Python

Alternatif 1: Tanpa menggunakan modul statistics

import math

# Fungsi untuk menghitung rata-rata
# Input: array
def mean(data):
   # μ = (Σ xi)/N
   return float(sum(data) / len(data))

# Fungsi untuk menghitung standar deviasi
# Input: array
def standar_deviasi(data):
   # D : Array dari deviasi
   # Untuk menyimpan deviasi setiap data
   # sehingga memungkinkan output nilai deviasi
   # setiap elemen data, jika diinginkan
   D = []
   rata_rata = mean(data)
   for x in data:
       # tambahkan nilai deviasi untuk setiap elemen data
       # Di = |xi – μ|
       D.append(abs(x-rata_rata))
   # Kembalikan nilai standar deviasi
   # σ = √[(Σ(Di)²)/N]
   # N diperoleh dari len(data)
   return math.sqrt(sum([d**2 for d in D])/len(data))

# Program Utama
if __name__ == '__main__':
   data = []
   # Loop forever untuk meminta input data
   # Berhenti ketika input data < 0
   while True:
       # Meminta masukan data bilangan
       x = int(input("Masukkan data ke-" + str(len(data)+1) + ": "))
       # Jika x > 0, masukkan ke array data
       if (x > 0):
           data.append(x)
       # Jika x = 0, ulangi input data
       elif (x == 0):
           print("Data harus merupakan bilangan bulat positif.")
       # Jika x < 0, keluar dari looping
       else: # x < 0
           break
   # Cetak data, rata-rata dan standar deviasi
   print("========================")
   print("Data:", end = " ")
   print(*data)
   print("RATA-RATA = " + str(mean(data)))
   print("STANDAR DEVIASI = " + str(standar_deviasi(data)), end="\n\n")
### END OF PROGRAM ###

10. Tekanan darah pasien satu, yang diukur setiap hari selama beberapa minggu, rata-ratanya adalah 182 dengan deviasi standar 5.3, sedangkan rata-rata tekanan darah pasien lainnya adalah 124 dengan deviasi standar 9.4. Tekanan darah pasien mana yang relatif lebih bervariasi?


Tekanan darah pasien kedua relatif lebih bervariasi karena memiliki standar deviasi yang lebih besar dibandingkan pasien pertama. Standar deviasi pasien kedua adalah 9.4 > 5.3.

Penjelasan dengan langkah - langkah:

Diketahui :

Tekanan darah pasien pertama yang diukur setiap hari selama beberapa minggu  rata-ratanya adalah 182 dengan standar deviasi 5,3. Sedangkan rata-rata tekanan darah pasien kedua adalah 124 dengan standar deviai 9,4.

Ditanya :

Tekanan darah pasien mana yang relative lebih bervariasi

Jawab :

Sebelum menganalisis soal terlebih dahulu kita ketahui hubungan antara rata-rata dengan standar deviasi itu sendiri. Standar deviasi adalah nilai yang dipakai untuk menentukan berapa dekatnya antara suatu data yang diteliti dengan mean ataupun rata-rata data tersebut. Jika nilai standar deviasi rendah maka nilai tersebut semakin mendekati rata-rata dari datanya. Apabila nilai standar deviasi tinggi berarti rentang variasinya semakin lebar.

Merujuk pada pengertian tersebut dan pertanyaan yang diminta soal, maka data yang lebih bervariasi adalah data pasien kedua dengan standar deviasi 9.4, karena nilai standar deviasinya lebih tinggi dari pasien pertama yaitu 5.3.

11. tolong yaa,,Sebuah sampel random berukuran 25 diambil dari suatu populasi normal yang mempunyai rata-rata 80 dan standar deviasi 5. Sampel random yang lain diambil dari populasi yang lain dengan ukuran 36, mempunyai rata-rata 75 dan standar deviasi 3. Hitunglah probabilitas bahwa rata-rata sampel pertama akan melampaui rata-rata sampel kedua dengan sekurang-kurangnya 3,4 tetapi kurang dari 5,9.​


Penjelasan dengan langkah-langkah:

Untuk menghitung probabilitas tersebut, kita perlu menggunakan distribusi sampling distribution of the difference between the means atau distribusi sampel perbedaan rata-rata.

Kita akan menggunakan rumus z-score untuk menghitung probabilitas tersebut:

z = (x̄1 - x̄2 - μd) / (σd / √n1 + n2)

dimana:

- x̄1 = rata-rata sampel pertama

- x̄2 = rata-rata sampel kedua

- μd = selisih rata-rata populasi (μd = μ1 - μ2)

- σd = standar deviasi perbedaan populasi (σd = √(σ1²/n1 + σ2²/n2))

- n1 = ukuran sampel pertama

- n2 = ukuran sampel kedua

Dalam kasus ini, kita ingin mencari probabilitas bahwa rata-rata sampel pertama akan melampaui rata-rata sampel kedua dengan sekurang-kurangnya 3,4 tetapi kurang dari 5,9. Dengan kata lain, kita ingin mencari P(3.4 ≤ x̄1 - x̄2 < 5.9).

Langkah-langkahnya adalah sebagai berikut:

1. Hitung selisih rata-rata populasi μd = μ1 - μ2 = 80 - 75 = 5

2. Hitung standar deviasi perbedaan populasi σd = √(σ1²/n1 + σ2²/n2) = √((5²/25) + (3²/36)) = 1.107

3. Hitung z-score menggunakan rumus di atas:

z = (x̄1 - x̄2 - μd) / (σd / √n1 + n2)

z = (3.4 - 0 - 5) / (1.107 / √(25 + 36))

z = -3.32

4. Hitung probabilitas menggunakan tabel distribusi normal standar atau kalkulator probabilitas:

P(z > -3.32) = 0.999

Jadi, probabilitas bahwa rata-rata sampel pertama akan melampaui rata-rata sampel kedua dengan sekurang-kurangnya 3,4 tetapi kurang dari 5,9 adalah sekitar 0.999 atau sekitar 99,9%.

18. Sebuah data memiliki rata-rata 80 dan standar deviasi 4,5. Apa makna dari data ini?


Jawab:

Standar deviasi mengukur seberapa jauh data tersebar dari rata-rata. Semakin besar standar deviasi, semakin besar juga variabilitas atau penyebaran data dari rata-rata. Dalam konteks ini, standar deviasi 4,5 menunjukkan bahwa sebagian besar data (sekitar 68%) berada dalam rentang sekitar 4,5 poin dari rata-rata (80). Dengan kata lain, sebagian besar data kemungkinan besar berada antara 75,5 (80 - 4,5) dan 84,5 (80 + 4,5).

27. Data berikut 4,6,10,6,12,14,8,4 tentukan rata rata dan standar deviasinya


Rata - rata dan standar deviasi dari data 4, 6, 10, 6, 12, 14, 8 dan 4 adalah 8 dan 3,46.

Simpangan baku atau deviasi standar (standard deviation) merupakan ukuran persebaran data atau jarak rata – rata penyimpangan antara nilai hasil pengukuran dengan nilai rata – rata. Dengan demikian, untuk menentukan sebuah simpangan baku sejumlah data, kita perlu menghitung rata – rata dari sejumlah data tersebut yang kemudian kita olah kembali dalam ragam atau varians dan berakhir dengan pengakaran varians tersebut menjadi sebuah simpangan baku.

Seperti yang telah kita pelajari, rata - rata dihitung dengan menjumlahkan seluruh data dan kemudian membaginya dengan frekuensi datanya (∑Xi / ∑f ).

Maka, standar deviasi dihitung dengan rumus : S = √(1/n [∑(xi - x)²]).

Untuk lebih jelasnya, kita akan ulas soal di atas dalam pembahasan di bawah ini.

PEMBAHASAN :

Diketahui sejumlah data 4, 6, 10, 6, 12, 14, 8 dan 4 akan dihitung nilai rata - rata dan  simpangan bakunya. Maka, menghitung rata – ratanya adalah dengan cara :

Rata – rata = ∑Xi / ∑f = (4 + 6 + 10 + 6 + 12 + 14 + 8 + 4) / 8

= 64 / 8

= 8

Selanjutnya, kita menghitung ragam atau variansnya.

S² = 1/n [∑(xi - x)²]

= ⅛ [(4 – 8)² + (6 – 8)² + (10 – 8)² + (6 – 8)² + (12 – 8)² + (14 - 8)² + (8 - 8)² + (4 - 8)²]

= ⅛ [(-4)² + (-2)² + (2)² + (-2)² + (4)² + (6)² + (0)² + (-4)²]

= ⅛ [16 + 4 + 4 + 4 + 16 + 36 + 0 + 16]

= ⅛ × 96

= 12

Dan terakhir, standar deviasi atau simpangan baku (S) dihitung dengan akar dari varians, maka standar deviasi data di atas adalah :

S = √S²

= √12

S = standar deviasi = √12

= 3,4641016151 ≈ 3,46

Jadi, rata - rata dan standar deviasi dari data 4, 6, 10, 6, 12, 14, 8 dan 4 adalah 8 dan 3,46.

40. suatu populasi memiliki rata-rata 75 dan standar deviasi 5. jika diambil sampel sebanyak 36, maka nilai rata-rata sampel tersebut mendekati


Nilai rata-rata sampel tersebut mendekat 75 untuk standar deviasi  [tex]\frac{5}{6}[/tex]. Soal tersebut merupakan soal tentang pengolahan dan penyajian data.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal Matematika yang membahas tentang pengolahan dan penyajian data. Untuk menyelesaikan soal tersebut kita harus menggunakan persamaan untuk mencari standar deviasi rata-rata sampel.

Persamaan standar deviasi rata-rata sampel

[tex]\frac{\sigma}{\sqrt{n} }[/tex]

Dengan:

σ = standar deviasin = banyak sampel

Penyelesaian soal

Diketahui:

σ = 5n = 36

Ditanyakan:
Tentukan nilai rata-rata sampel yang mendekati!

Jawab:

[tex]\frac{\sigma}{\sqrt{n} }[/tex][tex]\frac{5}{\sqrt{36} }[/tex] = [tex]\frac{5}{6}[/tex]

Jadi, nilai rata-rata sampel tersebut mendekat 75 untuk standar deviasi  [tex]\frac{5}{6}[/tex].
