14. PROGAM UNTUK MEMBUAT APLIKASI DEKSTOP


1. TSPlus

￼TSPlus

Aplikasi TSPlus adalah sebuah aplikasi dimana kamu bisa menggunakannya untuk membangun sebuah sistem Client Server dimana aplikasi yang kamu buat ada di Server, sebetulnya TSPlus punya cara kerja yang sama seperti Remote Desktop nya windows, namun dengan TSPlus kamu dapat mendifinisikan lebih mendetail mengenai User, Pemilihan Printer, kamu juga bisa membuat executable filenya dengan menggunakan TSPlus App Generator. Oh iya dengan TSPlus ini juga kamu bisa memilih Resolusi Aplikasi yang digunakan jika Bandwidth yang tersedia kecil pada client. Untuk mencoba software ini, kamu dikenakan biaya ya sob, alias software ini Berbayar, tapi kamu bisa coba kok versi trial nya disini.

2. 2X Application

￼2X Application Server

Tidak seperti TSPlus, 2X Application menggunakan port tersendiri untuk komunikasi antar client dan servernya,dengan menggunakan 2X Application kamu dapat meng hosting lebih dari 2 aplikasi pada sisi server, sehingga kamu bisa lebih leluasa dalam mempublish kan Aplikasi yang kamu buat. Berbeda dengan TSPlus, 2X Application memerlukan 2X Application client untuk dapat mengakses Aplikasi yang ada pada server. Untuk mencoba aplikasi ini kamu bisa dapatkan versi trial nya disini.

3. Winflector

￼Winflector

Software WinFlector adalah software yang boleh dibilang paling ringan diantara kedua software diatas, WinFlector bekerja seperti 2X Application yang membutuhkan Software yang terinstall pada sisi server, dan juga pada sisi Client untuk dapat mengakses aplikasi yang berada pada sisi server. Oh iya guys, Winflector sangat ringan digunakan dikoneksi yang boleh dibilang agak kurang cepat, dengan fitur TCP/IP Compress nya, membuat aplikasi dekstop kamu seperti halnya aplikasi yang dapat diakses dimanapun. kamu bisa coba versi gratis dari winflector disini, oh iya guys, versi gratis daripada winflector ini dibatasi 2 session saja. tapi so far, sangat berguna bagi mereka yang ingin membuat online sebuah program desktop menjadi online.

19. Apa saja kebutuhan hardware dan software yang perlu disiapkan dan dibutuhkan untuk membuat aplikasi berbasis Android jelaskan pula kegunaannya​


Jawaban:

1. Personal computer

Personal Computer atau PC merupakan perangkat utama dalam suatu jaringan computer. PC inilah yang akan bekarja mengirim dan mengakses data dalam jaringan.

2. NIC / LAN CARD

NIC merupakan port yang menghubungkan computer dengan jaringan. Port atau card ini bergabung di dalam PC yang menempel pada Main Board. Berdasarkan tipe bus, ada beberapa tipe NIC atau Network Card, yaitu ISA dan PCI.

3. Kabel jaringan

- Kabel Coaxial

Tampilan fisik terdiri dari kawat tembaga sebagai inti yang dilapisi oleh isolator.

- Fiber Optik

Terbuat dari serat kaca dengan teknologi canggih dan mempunyai kecepatan transfer data yang lebih cepat.

- Twisted Pair

Terdiri dari beberapa kabel yang saling melilit. Ada dua jenis kabel Twisted Pair. Shielded Twisted Pair (STP) dan Unshielded Twisted Pair (UTP).

4. Konektor RJ45

Sebagai saran penghubung antar kabel dengan colokan NIC.

5. Hub/Switch

Hub adalah alat yang digunakan untuk menghubungkan antar komputer dan perlatan lain dalam jaringan. Fungsi hub yaitu untuk mengarahkan informasi dalam jaringan sehingga sampai ke tujuan.

Apa itu software desain grafis?

Mengutip Computer Hope, software desain grafis adalah program perangkat lunak yang digunakan untuk membuat desain grafis. Desain grafis adalah representasi visual ide dengan simbol, kata, dan gambar pada komputer.

MelansirTech Radar, software desain grafis mengacu pada program atau kumpulan program yang memungkinkan untuk memanipulasi gambar atau model secara visual pada komputer.

Grafik komputer dapat diklasifikasikan menjadi kategori grafik raster dan grafik vektor, dengan varian 2D dan 3D. Ada program grafik yang berfokus pada grafik vektor atau raster saja. Ada yang bisa beroperasi pada keduanya.

Yang perlu diperhatikan, mudah untuk mengubah grafik vektor ke grafik raster, tetapi sulit untuk melakukan sebaliknya.

Saat ini banyak jenis software desain grafis, baik gratis maupun berbayar. Munculnya beragam perangkat lunak desain grafis ini sesuai kebutuhan dan preferensi pengguna yang berbeda.

GIMP adalah perangkat lunak pengedit gambar gratis dan sumber terbuka. Memiliki banyak fitur untuk mengedit gambar dengan mudah dari foto sederhana menjadi karya seni.

Pixlr

Pixlr adalah aplikasi pengeditan gambar berbasis web yang dapat dijalankan di browser dan sistem operasi apa pun. Aplikasi ringan dan menghabiskan sedikit memori bahkan di perangkat seluler.

Berfungsi sebagai alat pengeditan foto. Terdapat template format multigambar, lebih dari dua juta filter, overlay, dan efek.

Inkscape

Inkscape adalah perangkat lunak desain grafis yang kaya fitur dan sumber terbuka serta gratis. Mampu membuat vektor dengan mudah dan memiliki beragam alat pengeditan vektor.

Berfungsi untuk manipulasi aspek grafik, seperti bentuk, jalur, teks, klon, gradien, efek transparansi, dan lainnya. Dapat disimpan dalam format SVG, EPS, JPG, BMP, PNG, PostScript, atau TIP yang dapat diekspor ke format berbasis vektor.

Aplikasi Membuat Toko Online Android Dan iPhone Dengan MobiCartNov 12, 2014 2858

MobiCart adalah aplikasi yang disesuaikan untuk mobile e-commerce yang tidak hanya membuat tetapi juga untuk mengelolasebuah toko online mobile commerce pribadi Anda. Ada dua cara di mana MobiCart dapat bekerja. Yaitu Anda dapat memilih untuk menghubungkannya dengan situs yang ada atau membuat dengan MobiCart itu sendiri.

Jika Anda tidak memiliki website Anda sendiri, Anda tidak perlu khawatir karena MobiCart dapat menyelesaikan masalah tersebut sehingga Anda dapat memiliki toko aplikasi mobile Anda sendiri tanpa perlu terhubung dari sebuah situs web.

Tapi jika Anda sudah memiliki website Anda sendiri maka Anda memiliki pilihan untuk menggunakan API dari MobiCart dan memperbarui produk Anda langsung dari situs web Anda sendiri. Anda bahkan tidak perlu menggunakan admin MobiCart untuk memperbarui aplikasi toko online berbasis ponsel Anda.

Fitur MobiCart :

MobiCart memiliki kelebihan berupa pilihan kustomisasi penuh. MobiCart juga membantu dalam menambahkan produk dan memperbarui halaman untuk aplikasi Anda sendiri. MobiCart seperti toko pribadi Anda sendiri dengan nama merek pribadi Anda sendiri.Menjadi sepenuhnya berbasis web, MobiCart memberi Anda keuntungan memperbarui toko Anda kapan saja dan dari mana saja. Setiap kali Anda berusaha untuk memperbarui produk Anda dari situs anda maka secara bersamaan akan memperbarui toko ponsel pribadi Anda.MobiCart juga menyediakan dengan fitur Mobile e-commerce di mana Hal ini akan membantu Anda dalam bisnis Anda.MobiCart menyediakan fitur gratis. MobiCart memberikan kesempatan pada setiap orang untuk membuat serta menyebarkan toko e-commerce mobile mereka sendiri.MobiCart dapat menangani sejumlah pembelian yang dilakukan oleh sejumlah pelanggan dalam satu hari. Apakah itu aplikasi toko online untuk iPhone, Android, iPad atau bahkan telepon Blackberry. MobiCart tahu bagaimana menangani itu semua.Dengan Open API maka membuat dan mengalokasikan ekstensi untuk MobiCart sebagai add-on menjadi sangat mudah.
