8. Ini apa emang soalnya yang salah ya? Tolong bantu ya no 21. apa soalnya yg salah apa gimana


Tidak ada yg salah dengan soalnya kok... tulis reaksinya dulu masing2...

A) 2 H2(g) + O2(g) --> 2 H2O(l)
mol H2 = 1 / 2 = 0.5 mol
mol O2 = 16 / 32 = 0.5 mol
nah, padahal perbandingan koefisiennya seharusnya 1 : 2, tetapi perbandingan molnya 1 : 1, berarti ada yg salah dengan reaksi ini.

B) 2 Mg(s) + O2(g) --> 2 MgO(s)
mol Mg = 6/24 = 0.25 mol
mol O2 = 4/32 = 0.125 mol

reaksi ini sesuai dengan hukum kekekalan massa.

PERHATIAN UNTUK C DAN E: Sulfur di alam ada dalam allotrope yg berbeda-beda, tetapi untuk reaksi ini digunakan S2 dalam bentuk gas.

C) 2 Ca(s) + S2(g) --> 2 CaS(s)
mol Ca = 5 / 40 = 0.125 mol
mol S = 8 / 64 = 0.125 mol
Perbandingan koefisien 2:1 tetapi perbandingan mol 1:1, berarti pilihan C salah.

D) 3 Mg(s) + N2(g) --> Mg3N2(s)
mol Mg = 12 / 24 = 0.5 mol
mol N2 = 7 / 28 = 0.25 mol
perbandingan koefisien 3:1 tetapi perbandingan mol 2:1, berarti pilihan D salah.

E) 2 Mg(s) + S2(g) --> 2 MgS(s)
mol Mg = 3 / 24 = 0.125 mol
mol S2 = 8 / 64 = 0.125 mol
perbandingan koefisien 2:1 tetapi perbandingan mol 1:1, berarti pilihan E juga salah.

Nb: Mr S2 = 2 x 32 = 64, Mr N2 = 2 x 14 = 28, Mr O2 = 2 x 16 = 32, Mr H2 = 2 x 1 = 2

31. Jika dalam sebuah ulangan ada 15 soal pilihan ganda dan 5 soal essay, dan kamu hanya menjawab 14 soal pilihan ganda, dan 2 soal essay, kira kira berapa sih nilainya??


Perkiraan nilai ulangan dengan menjawab 14 soal pilihan ganda benar dan 2 soal essay benar adalah 6 atau 60. Untuk memperoleh nilai lebih besar, usahakan menjawab lebih banyak soal essay dengan benar.

NILAI ULANGAN

Operasi hitung bilangan dalam matematika menekankan urutan operasi dimana yang harus didahulukan adalah

Operasi di dalam tanda kurung Perkalian atau pembagian Penjumlahan dan pengurangan

Operasi matematika ini sebenarnya dipergunakan para guru untuk menentukan nilai ulangan berdasarkan jumlah soal dan bentuk soal yang diberikan. Jika

30 soal pilihan ganda maka setiap benar mendapatkan nilai 1
Untuk skala nilai 0 - 10
Hasil nilai = [tex]jumlah \: nilai \: benar \div 3[/tex]
Untuk skala nilai 0 - 100
Hasil nilai = [tex](jumlah \: nilai \: benar \div 3) \times 10[/tex] 15 soal pilihan ganda dan 5 essay maka setiap benar untuk soal pilihan ganda mendapatkan nilai 2 dan setiap benar untuk soal essay mendapatkan nilai 4.
Untuk skala nilai 0 - 10
Hasil nilai = [tex]jumlah \: nilai \: benar \div 5[/tex]
Untuk skala nilai 0 - 100
Hasil nilai = [tex](jumlah \: nilai \: benar \div 5) \times 10[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

15 soal pilihan ganda dan 5 essay Menjawab benar 14 soal pilihan ganda dan 2 soal essay

Ditanyakan:

Nilai?

Jawaban:

Menentukan jumlah nilai

Jumlah nilai = [tex](benar \: PG \times 2) \:+\: (benar \: essay \times 4)[/tex]

= [tex](14 \times 2) \:+\: (2 \times 4)[/tex]

= [tex]28 \:+\: 8[/tex]

= 36

Hasil nilai

Untuk skala nilai 0 - 10
Hasil nilai = [tex]36 \div 6[/tex]
Hasil nilai = 6 Untuk skala nilai 0 - 100
Hasil nilai = [tex](36 \div 6) \times 10[/tex]
Hasil nilai = [tex]6 \times 10[/tex]
Hasil nilai = 60

39. gimana la tolong soalnya buat belajar uas​


Jawab:

Penjelasan dengan langkah-langkah:

( 2p²qr⁻³/4pqr⁻² )²

= ( p/2r)²

= p²/4r²

atau

(p²r⁻²)/4

semoga bisa membantu

Jawaban:

Bentuk sederhana dari

[tex] = (\frac{{2p}^{2}q {r}^{ - 3} }{4pq {r}^{ - 2} }) {}^{2} [/tex]

[tex] = ( \frac{ {2}^{2} {p}^{(2 \times 2)} {q}^{2} {r}^{ (- 3 \times 2)} }{ {4}^{2} {p}^{2} {q}^{2} {r}^{( - 2 \times 2)} } )[/tex]

[tex] = ( \frac{4 {p}^{4 } {q}^{2} {r}^{ - 6} }{16 {p}^{2} {q}^{2} {r}^{ - 4} } )[/tex]

[tex] = \frac{ {p}^{(4 - 2)} {q}^{(2 - 2)} {r}^{( - 6 - ( - 4)} }{4} [/tex]

[tex] = \frac{ {p}^{2} {r}^{( - 6 + 4)} }{4} [/tex]

[tex] = \frac{ {p}^{2} {r}^{ - 2} }{4} [/tex]

jadi, bentuk sederhana nya adalah :

[tex] = \frac{ {p}^{2} {r}^{ - 2} }{4} [/tex]
