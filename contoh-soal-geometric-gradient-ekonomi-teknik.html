34. jumlah n suku pertama suatu deret adalah 2n²-n.Deret tersebut adalah....A.Bukan deret aritmatika dan bukan deret geometricB.Deret geometric dengan rasio 2C.Deret aritmatika dengan beda 2D.Deret geometric dengan rasio 4E.Deret aritmatika dengan beda 4​


Jawaban:

E. Deret aritmetika dengan beda 4

Penjelasan dengan langkah-langkah:

Gunakan hubungan antara Un dengan Sn sebagai berikut:

Un = Sn - Sn-1

= 2n² - n - [2(n - 1)² - (n - 1)]

= 2n² - n - [2(n² - 2n + 1) - n + 1]

= 2n² - n - [2n² - 4n + 2 - n + 1]

= 2n² - n - [2n² - 5n + 3]

= 2n² - n - 2n² + 5n - 3

= 4n - 3

= 4n - 4 + 1

= 4(n - 1) + 1

Perhatikan bahwa bentuk di atas mirip dengan suku ke-n pada barisan aritmetika, yaitu Un = a + (n - 1)b. Berdasarkan bentuk di atas, diperoleh a = 1 dan b = 4, sehingga deret tersebut adalah deret aritmetika dengan beda 4.
